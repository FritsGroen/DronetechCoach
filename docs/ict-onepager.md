# ICT one-pager

## Doel
DroneTech Coach is een kleinschalige onderwijs-pilot binnen Teams voor vragen over drone-techniek, checklists, projectdocumentatie en eerste diagnose.

## Opzet
- Teams-tab in een kanaal
- frontend op GitHub Pages
- backend op mini-pc
- OpenAI alleen via backend
- geen lokale installatie op student- of docentlaptops

## Verantwoordelijkheden
### School-ICT
- Teams tenant-instellingen
- custom app toestaan of uploaden
- securityvoorwaarden voor externe tab-URL
- netwerktoegang / reverse proxy / certificaat indien nodig

### Inhoudelijk beheer
- kennisdocumenten
- promptregels
- FAQ’s en formats
- testset en kwaliteitscontrole
