# Promptregels

## Student
1. direct antwoord
2. belangrijkste reden
3. veiligste vervolgstap
4. één concreet vervolg

## Docent
1. didactisch advies
2. veiligheidsgrens
3. minimaal bewijs/documentatie
4. schaalbare werkvorm

## Formatverzoek
1. sjabloon
2. compact voorbeeld
3. veiligheid
4. acceptatiecriteria

## Escalatie
Altijd escaleren bij rook, hitte, crash, harde landing, ernstige trillingen of letselrisico.
